# 설치 방법 (네이버 동영상 다운로더)

크롬 웹스토어를 거치지 않고 직접 설치하는 방식입니다. 크롬·웨일·엣지 모두 동일합니다.

## 처음 설치하기

1. 받은 **zip 파일의 압축을 풉니다.** (예: `바탕화면\naver-video-downloader\`)
   - 폴더 안에 `manifest.json` 이 보이면 제대로 푼 것입니다.
   - ⚠️ 이 폴더는 **지우거나 옮기면 안 됩니다.** 확장이 이 폴더를 직접 참조합니다.
2. 브라우저 주소창에 아래를 입력해 확장 관리 페이지를 엽니다.
   - 크롬: `chrome://extensions`
   - 웨일: `whale://extensions`
   - 엣지: `edge://extensions`
3. 오른쪽 위(엣지는 왼쪽) **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드** 버튼을 누르고, 1번에서 압축 푼 **폴더**를 선택합니다.
5. 툴바에 아이콘이 생기면 완료입니다. 네이버 블로그 글에서 아이콘을 누르면 오른쪽에 패널이 열립니다.

> 아이콘이 안 보이면 툴바의 퍼즐(확장) 아이콘을 눌러 고정(핀)하세요.

## 업데이트 하기

확장이 새 버전을 발견하면 패널 위에 **"새 버전 있음 · 클릭해서 다운로드"** 배너가 뜹니다.

1. 배너를 클릭해 새 zip을 받습니다.
2. **기존 폴더에 덮어써서** 압축을 풉니다. (폴더 위치는 그대로 유지)
3. 확장 관리 페이지(`chrome://extensions`)에서 이 확장의 **새로고침(↻)** 버튼을 누릅니다.

> 폴더 위치를 그대로 두고 덮어쓰면, 로드했던 확장이 새 코드로 갱신됩니다.

## 사용법

1. 네이버 블로그·카페·포스트 등 영상이 있는 글에 들어갑니다. (자동 감지 — 새로고침 불필요)
2. 툴바 아이콘 클릭 → 오른쪽에서 패널이 슬라이드로 열립니다.
3. 썸네일 목록에서 화질(720p 등) 선택 → 다운로드 버튼(⬇) 클릭.
4. `다운로드/NaverVideo/` 폴더에 저장됩니다.
5. 패널 바깥 클릭 · ESC · 닫기(✕)로 닫힙니다.

## 참고

- 감지가 안 되면 패널 우측 상단 **다시 검색(↻)** 을 누르거나 영상을 한 번 재생해 보세요.
- 영상 주소는 네이버가 서명한 임시 링크(약 8시간)라, 감지되면 바로 받는 것이 안전합니다.
- 본인이 시청 권한을 가진 콘텐츠의 개인 소장 용도로만 사용하세요.

# 네이버 동영상 다운로더 (웨일/크롬 확장)

네이버 블로그·카페·포스트 페이지에 들어가면 **자동으로** 동영상을 감지해서,
썸네일 목록 + 화질 선택 + 원클릭 다운로드를 제공하는 글래스모피즘 UI 확장입니다.
툴바 아이콘을 누르면 화면 **오른쪽 끝에서 슬라이드로 나타나는 드로어 패널**이 열리고,
패널 바깥을 클릭하거나 ESC를 누르면 다시 슬라이드로 닫힙니다.

## 동작 원리

1. **자동 감지 (새로고침 불필요)**
   - 페이지 로드 시작 시점(`document_start`)부터 페이지의 `fetch`/`XHR`을 후킹해,
     플레이어가 받아오는 VOD 응답(`?key=` MPD manifest / 재생 API JSON)을 그대로 가로챕니다.
   - 추가로 본문(스마트에디터 모듈)에서 `vid`/`inkey`를 찾아 재생 버튼을 누르지 않아도
     미리 영상 정보를 조회합니다.
2. **파싱** — MPD의 `Representation`별 `BaseURL`(화질별 mp4 주소), 썸네일, 제목, 길이를 추출합니다.
3. **다운로드** — 팝업에서 화질 칩을 선택하고 다운로드 버튼을 누르면 페이지 이동 없이
   `chrome.downloads` API로 바로 저장됩니다. (`다운로드/NaverVideo/제목_720p.mp4`)

## 설치 (웨일 브라우저)

1. 웨일 주소창에 `whale://extensions` 입력
2. 우측 상단 **개발자 모드** 켜기
3. **압축해제된 확장앱 설치** 클릭
4. 이 폴더(`E:\개발\naverVideo`) 선택

크롬/엣지도 동일합니다 (`chrome://extensions` / `edge://extensions`).

## 사용법

1. 네이버 블로그 글 등 영상이 있는 페이지에 들어갑니다.
2. 툴바 아이콘에 감지된 영상 개수가 배지로 표시됩니다.
3. 아이콘 클릭 → 오른쪽 끝에서 **드로어 패널**이 슬라이드로 열립니다.
4. 썸네일 목록에서 화질(720p 등) 선택 → 다운로드 버튼 클릭.
5. 패널을 열어둔 상태에서 그 페이지의 영상이 추가로 감지되면 목록이 실시간 갱신됩니다.
6. 패널 **바깥 클릭 · ESC · 닫기(✕)** 버튼으로 슬라이드 아웃하며 닫힙니다.
7. 감지가 안 되면 패널 우측 상단 **다시 검색**(↻) 버튼을 누르거나 영상을 한 번 재생하세요.

## 파일 구조

```
manifest.json            MV3 매니페스트
background.js            탭별 영상 목록 저장, 배지, 다운로드 처리
content/interceptor.js   (MAIN world) fetch/XHR 후킹 + vid/inkey 선제 조회
content/bridge.js        (isolated) MPD/JSON 파싱·정규화 → 백그라운드 전달
content/panel.js         (isolated) 오른쪽 슬라이드 드로어 주입·애니메이션·바깥클릭 닫기
popup/popup.html|css|js  글래스모피즘 패널 UI (드로어 iframe으로 로드)
popup/preview.html       개발용 미리보기 (chrome API mock, 배포 zip에서 제외됨)
icons/                   확장 아이콘
build.ps1                배포용 클린 zip 생성 (개발 파일 제외)
INSTALL.md               최종 사용자(친구)용 설치 안내서
```

## 배포 & 자동 업데이트

크롬 웹스토어를 거치지 않고 **압축해제 설치(Load unpacked)**로 배포합니다.

1. `.\build.ps1` 실행 → `dist\naver-video-downloader-v<버전>.zip` 생성 (`.claude`·`preview.html` 등 개발 파일 자동 제외).
2. 이 zip과 [INSTALL.md](INSTALL.md)를 함께 공유하면, 받는 사람은 개발자 모드에서 폴더를 로드해 설치합니다.

**업데이트 확인기** — 크롬은 웹스토어 밖 확장의 조용한 자동 업데이트를 막으므로, 대신 확장이
12시간마다 GitHub 최신 릴리스를 확인해 새 버전이 있으면 패널에 배너를 띄웁니다.

- [background.js](background.js) 상단의 `const REPO = 'YOUR_GITHUB_ID/naver-video-downloader'` 를
  본인 저장소로 바꾸세요. (GitHub API는 CORS 허용 → 추가 권한/경고 없음)
- 새 버전 배포 절차: `manifest.json`의 `version` 올리기 → `build.ps1` 실행 →
  GitHub에서 **Release**(태그 `v1.0.1` 등) 생성하고 zip 첨부.
- 설치된 확장들은 배너 → 클릭 → 새 zip 다운로드 → 기존 폴더 덮어쓰기 → 확장 새로고침(↻) 흐름으로 갱신됩니다.

## 참고 사항

- 영상 URL은 네이버가 서명한 임시 주소(약 8시간 유효)라 감지 직후 바로 받는 것이 안전합니다.
- HLS(mp2t) 스트림은 제외하고 직접 저장 가능한 **mp4 화질만** 목록에 표시합니다.
- 드로어는 페이지 위에 떠 있는 오버레이라, 다른 글로 이동하면 패널이 닫힙니다. 새 페이지에서 아이콘을 다시 누르면 됩니다.
- 탭을 닫거나 다른 페이지로 이동하면 감지된 목록도 초기화됩니다.
- 본인이 시청 권한을 가진 콘텐츠의 개인 소장 용도로만 사용하세요.
